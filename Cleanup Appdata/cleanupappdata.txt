Docker Not Running=Docker neběží
Because the docker service is not currently running, this module will prompt you to delete the appdata for any/all docker applications which are installed=Protože služba Docker momentálně neběží, tento modul ti nabídne možnost smazat složky appdata všech nainstalovaných Docker aplikací.

Deleted=Smazáno
The following appdata paths have been determined based upon your previously installed applications to be no longer in use Select which ones you would like to be deleted=Následující složky appdata byly na základě dříve nainstalovaných aplikací vyhodnoceny jako nepoužívané. Vyber, které chceš smazat.
Note that this module reviews all of your previously installed Docker Containers to determine which appdata folders are abandoned. If you deleted the previously installed list (either through Community Applications or manually), those results will not be displayed. In other words, if there is no record for a folder within the previously installed Containers, you will not be prompted to delete it.=Upozornění: Tento modul prochází všechny dříve nainstalované Docker kontejnery a zjišťuje, které appdata složky jsou opuštěné. Pokud jsi seznam dříve nainstalovaných aplikací smazal (přes Community Applications nebo ručně), tyto výsledky se nezobrazí. Jinými slovy: pokud neexistuje záznam o složce v dříve nainstalovaných kontejnerech, nebude ti nabídnuta ke smazání.
In the case of a severely misconfigured previously installed app, the entire contents of a share may be proposed for deletion. It is your responsibility to confirm the folders proposed for deletion. If you inadvertently delete a share you intended to keep due to misconfiguration, the deletion is not reversible and the plugin author assumes no responsibility=V případě vážně chybně nakonfigurované dříve nainstalované aplikace může být navržen ke smazání celý obsah sdíleného úložiště. Je tvojí odpovědností potvrdit nabízené složky ke smazání. Pokud omylem smažeš sdílení, které sis chtěl ponechat kvůli chybné konfiguraci aplikace, smazání nelze vrátit zpět a autor pluginu za to nenese žádnou odpovědnost.
